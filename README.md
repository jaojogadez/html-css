# html-css

Curso de HTML5 e CSS3 do Curso em Vídeo.

## Description

This repository contains a collection of HTML5 and CSS3 projects and exercises completed as part of the "Curso de HTML5 e CSS3" from Curso em Vídeo. It covers fundamental concepts and practical applications of web development technologies.

## Features and Functionality

The repository showcases various HTML and CSS features, including:

*   **HTML Structure:** Semantic HTML, headings, paragraphs, lists, links, images, and tables.
*   **CSS Styling:** Inline, internal, and external stylesheets; selectors, properties (colors, fonts, backgrounds, box model), pseudo-classes, and pseudo-elements.
*   **Media Handling:** Embedding images, audio, and video. Responsive images using the `<picture>` element.
*   **Form Creation:** Designing and implementing HTML forms with various input types, validation, and attributes.
*   **Layout Techniques:** Flexbox for creating flexible and responsive layouts.
*   **Responsive Design:** Media queries for adapting websites to different screen sizes and orientations.
*   **Tables:** Table creations with merged cells, scope and responsive tables.

## Technology Stack

*   HTML5
*   CSS3

## Prerequisites

*   A modern web browser (e.g., Chrome, Firefox, Safari, Edge)
*   A text editor or IDE (e.g., VS Code, Sublime Text, Atom)

## Installation Instructions

No installation is required to view the HTML and CSS files. Simply clone or download the repository to your local machine:

```bash
git clone https://github.com/jaojogadez/html-css
```

## Usage Guide

Navigate to the desired project or exercise directory within the repository. Open the HTML file in your web browser to view the result.

### Example: Viewing the Android Mascot History

1.  Navigate to the `desafios/desafio10 - aula/` directory.
2.  Open `android.html` in your web browser.
